Supported Elements

The assertEquals command can currently be used on any HTML element. There are currently no checks. However, you should avoid using the command on elements that cannot legally have a <span> child (e.g. <table>, <tr>, <ul>, <ol>).

Examples

All of these will work:

<span concordion:assertEquals="#name">Fred</span>

<strong concordion:assertEquals="#name">Fred</strong>

<div concordion:assertEquals="#name">Fred</div>

<table>
    <tr>
        <td concordion:assertEquals="#name">Fred</td>
    </tr>
</table>
