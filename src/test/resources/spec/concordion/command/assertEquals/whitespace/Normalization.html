Normalization

Whitespace characters are:

  • Tab (ASCII 9)
  • LF (ASCII 10)
  • CR (ASCII 13)
  • Space (ASCII 32)

Examples

Source String Normalized
fred fred
[SPACE]fred fred
[SPACE]fred[SPACE] fred
[SPACE][SPACE]fred[LF][SPACE] fred
[SPACE][CR][LF][TAB][SPACE]fred[SPACE][SPACE]bloggs fred[SPACE]bloggs
fred[LF][SPACE][TAB][CR][LF]bloggs fred[SPACE]bloggs
fred[LF][SPACE]x[TAB][CR][LF]bloggs fred[SPACE]x[SPACE]bloggs
