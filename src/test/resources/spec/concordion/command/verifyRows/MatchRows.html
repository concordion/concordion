verifyRowsUnordered

When used on a <table>, the verifyRowsUnordered command compares the contents of the table with the contents of a collection, and reports similarities, differences and missing or surplus rows.

The expression must return something Iterable (e.g. a Collection).

Example

<table concordion:matchRows="#username : usernames">
    <tr>
        <th concordion:assertEquals="#username">Username</th> 
    </tr>
    <tr>
        <td>bpeep</td> 
    </tr>
    <tr>
        <td>jspratt</td> 
    </tr>
    <tr>
        <td>jspratt</td>
    </tr>
</table>
usernames Collection Results in
bpeep, jspratt, jspratt bpeep->SUCCESS, jspratt->SUCCESS, jspratt->SUCCESS
jspratt, bpeep, jspratt bpeep->SUCCESS, jspratt->SUCCESS, jspratt->SUCCESS
bpeep, jspratt bpeep->SUCCESS, jspratt->SUCCESS, jspratt->MISSING
jspratt, bpeep bpeep->SUCCESS, jspratt->SUCCESS, jspratt->MISSING
jspratt, bpeep, bpeep bpeep->SUCCESS, jspratt->SUCCESS, jspratt->MISSING, bpeep->SURPLUS
bpeep, jspratt, jspratt, jspratt bpeep->SUCCESS, jspratt->SUCCESS, jspratt->SUCCESS, jspratt->SURPLUS
hdumpty, jspratt, jspratt bpeep->MISSING, jspratt->SUCCESS, jspratt->SUCCESS, hdumpty->SURPLUS
bpeep bpeep->SUCCESS, jspratt->MISSING, jspratt->MISSING
jspratt bpeep->MISSING, jspratt->SUCCESS, jspratt->MISSING
ppan, bpeep, jspratt bpeep->SUCCESS, jspratt->SUCCESS, jspratt->MISSING, ppan->SURPLUS
rhood, jspratt, ppan, mdaw bpeep->MISSING, jspratt->SUCCESS, jspratt->MISSING, rhood->SURPLUS, ppan->SURPLUS, mdaw->SURPLUS
