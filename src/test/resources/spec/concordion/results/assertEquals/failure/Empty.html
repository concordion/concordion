Empty Elements

If the element is empty, then a non-breaking space (&#160) is inserted (so that the failure shows up when displayed in a browser).

Example: Empty Expected

<div concordion:assertEquals="acronym" />

When marked as a failure, with actual value XYZ, it becomes: 

<div concordion:assertEquals="acronym" class="failure"> _
<del class="expected">&#160;</del>
<ins class="actual">XYZ</ins> _
</div>

Example: Empty Actual

<div concordion:assertEquals="acronym">ABC</div>

When marked as a failure, with a blank actual value, it becomes:

<div concordion:assertEquals="acronym" class="failure"> _
<del class="expected">ABC</del>
<ins class="actual">&#160;</ins> _
</div>
