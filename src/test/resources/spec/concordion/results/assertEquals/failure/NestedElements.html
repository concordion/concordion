Nested Elements

Nested elements are moved into the 'expected' span.

Example

<div concordion:assertEquals="acronym">My <em>simple</em> example</div>

When marked as a failure, with actual value XYZ, it becomes:

<div concordion:assertEquals="acronym" class="failure"> _
<del class="expected">My <em>simple</em> example</del>
<ins class="actual">XYZ</ins> _
</div>
