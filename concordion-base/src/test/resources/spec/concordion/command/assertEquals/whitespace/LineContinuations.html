Line Continuations

It is sometimes useful to be able to break a long line of text over several lines without the linebreaks being treated as whitespace.

You can do this by putting an underscore at the end of a line, preceded by a space.

Example

These two statements are treated identically:

(1) 
<pre concordion:assertEquals="#firstName">Fred Flintstone</pre>
(2) 
<pre concordion:assertEquals="#firstName">Fred Flint _
stone</pre>
