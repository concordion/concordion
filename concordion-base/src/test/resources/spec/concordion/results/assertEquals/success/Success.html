Success

Successes are indicated by adding a class="success" attribute to the element. Typically this will result in the element being displayed in green (depending on the CSS stylesheet).

Example

<span concordion:assertEquals="username">fred</span>

When marked as a success becomes:

<span concordion:assertEquals="username" class="success">fred</span>
